# Titanic Survival Prediction

📌 Overview
This project predicts whether a passenger survived the Titanic disaster using a Machine Learning model.  
The model analyzes passenger features such as **Age, Gender, Passenger Class, and Fare** to determine the probability of survival.

---
🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---
⚙️ Project Workflow
1. Data Loading and Exploration  
2. Handling Missing Values  
3. Data Cleaning and Preprocessing  
4. Feature Encoding  
5. Train-Test Split  
6. Logistic Regression Model Training  
7. Model Evaluation  

---
📊 Model Accuracy
The **Logistic Regression** model achieved an accuracy of **79.8%** on the test dataset.
